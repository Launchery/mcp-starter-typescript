# Roadmap

> Starter template for building MCP servers in TypeScript.

## v1.0 — Foundation ✅
- [x] MCP server with stdio transport
- [x] 3 example tools (echo, add, search)
- [x] TypeScript strict mode
- [x] Multi-client quickstart (Claude, Cursor)

## v1.1 — Polish
- [ ] GitHub Actions CI (build + lint)
- [ ] npm publishing workflow
- [ ] Resource support (in addition to tools)
- [ ] Prompt templates support
- [ ] Environment variable validation (zod)

## v1.2 — Examples
- [ ] Example: API documentation server
- [ ] Example: Database query server
- [ ] Example: File system operations server
- [ ] Example: HTTP API proxy server

## v2.0 — Advanced
- [ ] SSE transport support (in addition to stdio)
- [ ] Authentication middleware
- [ ] Rate limiting
- [ ] Logging middleware
- [ ] Tool composition patterns
